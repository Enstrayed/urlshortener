Create a Shortened URL

1. Enter your API Key under Authorization
2. Either press 'Random' to create a random URL or enter one yourself
3. Enter the target URL under 'Value' or press 'Clipboard' to have it pulled from your clipboard
4. Press 'POST' to submit the shortened URL
- If the request succeeds, the result field will be updated with the shortened URL
- If the Result field says "409", then the shortened URL already exists, press 'Re-Random & POST' to retry

Delete a Shortened URL

1. Enter your API Key under Authorization
2. Enter the shortened URL (without etyd.cc/) under 'URL'
3. Press 'DELETE'
- If the request succeeds, the result field will be updated to "200"
- If the Result field says "400", the entered short URL does not exist
